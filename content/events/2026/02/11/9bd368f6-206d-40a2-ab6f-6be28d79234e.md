## Bern Data Science Bootcamp

Learn data science by doing! In this intensive bootcamp, you will work through a complete data science project using Swiss open data — from raw data to a deployed prediction model.

### Project: Predicting Public Transport Delays

Using data from the Swiss Federal Railways (SBB), you will:

1. **Explore** — Data cleaning and exploratory analysis
2. **Engineer** — Feature engineering and selection
3. **Model** — Train and evaluate ML models
4. **Deploy** — Build a simple prediction API

### What to Bring

- Laptop with Python 3.10+
- Basic Python knowledge
- Enthusiasm for data!

Lunch, coffee, and all materials included.
